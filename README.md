# HabitTrack 🎯

HabitTrack es una aplicación web (SPA) diseñada para el seguimiento y la gamificación de rutinas diarias. Este prototipo ha sido desarrollado como Proyecto de Fin de Ciclo (TFG) para el grado de Desarrollo de Aplicaciones Web (DAW).

## 🚀 Tecnologías utilizadas
* **Frontend:** React (v18)
* **Estilos:** Tailwind CSS
* **Build Tool:** Vite
* **Almacenamiento:** Web Storage API (LocalStorage)

## ⚙️ Instalación y despliegue local

Para ejecutar este proyecto en tu propia máquina, asegúrate de tener [Node.js](https://nodejs.org/) instalado y sigue estos pasos:

1. Clona este repositorio:
   `git clone https://github.com/AlejandroLuisReyDEV/habit-track.git`

2. Accede al directorio del proyecto:
   `cd habit-track`

3. Instala las dependencias:
   `npm install`

4. Inicia el servidor de desarrollo:
   `npm run dev`

## 📖 Guía básica de uso
* **Crear un hábito:** Haz clic en el botón flotante inferior derecho, elige un nombre, un icono y un color representativo.
* **Marcar progreso:** En la vista principal, haz clic sobre los cuadros de los días para alternar el estado (completado/no completado). Los datos se guardan automáticamente en tu navegador.
* **Navegación:** Utiliza el menú para alternar entre las vistas semanal, mensual y anual (mapa de calor).
* **Modo Oscuro:** Puedes alternar el tema visual desde la sección de ajustes o el perfil de usuario.
